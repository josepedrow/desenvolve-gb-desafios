# Formulário Simples

### Nível

Fácil

### Objetivos

1. Centralizar todo o conteúdo na página;
2. Acrescentar validação no campo de e-mail (ao digitar um e-mai fora do padrão e clicar no botão de cadastrar, um erro do HTML5 deve ser exibido).

### Como enviar o desafio

1. Faça um fork do repositório (vídeo com tutorial no [YouTube](https://www.youtube.com/watch?v=l1rwvDvD1og));
2. Crie uma branch com o padrão `seu-usuario-git/nome-do-desafio`, exemplo: `matheusrufinos/formulario-simples`;
3. Crie uma pasta dentro da pasta do desafio com seu usuário do GitHub e copie os arquivos originais para dentro dela, exemplo: `formulario-simples/matheusrufinos`;
4. Faça as alterações necessárias no código e realize o commit + push na sua branch;
5. Abra uma Pull Request apontando da sua branch para branch main, descreva quais alterações foram realizadas e anexe um screenshot da solução final.
